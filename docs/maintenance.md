# 维护规范

## 命名规范

- 人物目录：`people/<person-slug>/`。
- 证据 ID：`E` + 三位或更多数字，例如 `E001`。
- 论文 ID：优先使用 DOI 后缀的简化形式；没有 DOI 时使用 `year-first-author-keywords`。
- 截图：`YYYYMMDD-short-description.png`。

## 新增人物检查清单

- [ ] 人物页已创建。
- [ ] 至少一条视频线索已记录。
- [ ] 关键结论已定位到论文图号、表号、补充材料文件或原始数据列。
- [ ] 论文条目已创建或链接。
- [ ] 截图有来源、时间和说明。
- [ ] CSV 数据已同步更新。
- [ ] 待核实内容已明确标注。

## 表述规范

- 使用「论文 Fig. X 显示」「补充表 Sx 的第 y 列显示」「耿同学视频 xx:xx 指出」等可追溯表述。
- 可以明确记录“造假/伪造/图片重复”等判断，但必须紧跟论文内证据定位。
- 对尚未回到论文原文核对的线索标注「待核实」，不要混入已实锤列表。
